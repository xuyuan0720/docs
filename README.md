趁世界还不那么拥挤。<br/>

去见你想见的人吧！<br/>
趁现在自己的双手还能拥抱彼此。<br/>
趁我们还有呼吸。<br/>

趁自己还活着。<br/>
趁他还活着。<br/>

就一次。<br/>
